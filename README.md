# Desert Camel Runner (2D)

A smooth and polished 2D endless runner game where a camel runs through the desert.

## Features
- 2D HTML5 Canvas
- Animated running camel
- Parallax-style desert background
- Mobile landscape optimized
- Instant input response
- Increasing difficulty
- Pause & restart

## Controls
- Tap screen or press Space to jump
